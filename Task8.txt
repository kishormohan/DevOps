Create a dockerfile, file which when build must display your basic details in website:

ubuntu@ip-172-31-84-167:~$ cat Dockerfile
From nginx
RUN apt update
COPY index.html /usr/share/nginx/html
EXPOSE 8005
CMD ["nginx","-g","daemon off;"]
